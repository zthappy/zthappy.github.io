title: Javascript 数组循环比较
date: 2017-07-12 16:49:33

tags:

for循环

最简单最通用的循环，代码如下：
for(var i = 0; i < arr.length;i++) {
// 代码
}

优化的for循环，代码如下：

for(var i = 0, len = arr.length;i<len;i++) {
    // 代码
}

for..in 循环

for(var i in arr) {
    // 代码
}

forEach循环，ie中无该方法，代码如下：

arr.forEach(function(value,index,arr){
    // 代码
})

map 循环

arr.map(function(value,index,arr) {
    // 代码
})

for..of循环 es6

for(var i of arr) {
    // 代码
}
