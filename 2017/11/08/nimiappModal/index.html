小程序的弹框

小程序的弹框对于用户的需求封装的已经很完善了，但是除了能修改按钮颜色之外，其他颜色都不能修改；
所以在我们设计妹妹的催促下，我们开始自定义弹框开发。

小程序的模板

在小程序里说到复用，首先想到的便是模板
定义模板


    <template name="msgItem">
      <view>
        <text> :  </text>
        <text> Time:  </text>
      </view>
    </template>

使用模板


    <template is="msgItem" data=""/>

但是模板的缺点在于：真正实现复用的只是页面部分，js还需要在每个页面重写。

使用模板实现真正的复用弹框

新建一个文件夹，新建wxml, wxss, js文件

在wxml中写模板弹框


    <template name="popmodel">
        <view class="mask" wx:if="isshowModel">
            <view class="modelDetail">
               <view class="title">title</view>
                <view class="smalltitle" wx:if="smalltitle">smalltitle</view>
                <view class="content" wx:if="content">content</view>
                <view class="btngroup">
                    <view class="btncancel" bindtap="cancel" wx:if="cancelText">cancelText</view>
                    <view class="btnconfirm" bindtap="confirm" wx:if="confirmText">confirmText</view>
                </view>
            </view>
        </view>
    </template>

js中实现弹框的显示隐藏，以及回调等功能


    var cpmdata = {
        title:"",
        content:"",
        isshowModel:false
    }
    var popModel = {
        showModel:function(title,smalltitle,content,cancelText,confirmText,conFn){
            var scope = this;
            console.log(confirmText);
            this.setData({
                title:title,
                smalltitle:smalltitle,
                content:content,
                cancelText:cancelText,
                confirmText:confirmText,
                conFn:conFn,
                isshowModel: true
            })
        },
        cancel:function(){
            this.setData({
                isshowModel:false
            })
        },
        confirm:function() {
            if(this.data.conFn){
                this.data.conFn();
            }
            this.setData({
                isshowModel:false
            })
        }

    }
    function PopModel(){
        var pages = getCurrentPages();
        var curPage = pages[pages.length - 1];
        this._page = curPage;
        Object.assign(curPage,popModel);
        curPage.popModel = this;
        curPage.setData(cpmdata);
        return this;
    }
    module.exports={
        PopModel : PopModel
    }

关键的实现在于：
将其全局化，在公共css中引用

 @import "/page/train/webapp/component/toastAndPop.wxss";

js 全局化

    module.exports = {
        ToastPannel : toastModel.ToastPannel,
        PopModel : popModel.PopModel,
    }

调用自定义弹框

wxml中跟原先一样引用模板
js中，在onload中引用 

commonjs.PopModel
，而后在需要使用弹框的地方，直接
this.showModel(…);
